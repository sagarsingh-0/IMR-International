# IMR International - Smart Digital Innovation

IMR International is a cutting-edge educational platform dedicated to Smart Digital Innovation. This project serves as a comprehensive portal for students to explore professional education, certification programs, and career-focused training.

## 🚀 Key Features
- **Smart Education Portal**: Access to BBA, BCA, and BSc Data Science programs.
- **AI-Powered (Local) ChatBot**: Instant answers to student queries about admissions, fees, and programs.
- **Dynamic Content Management**: Fully manageable admin dashboard for tracking applications and analytics.
- **Privacy First**: Built-in anonymous analytics with full user consent controls.
- **Seamless UX**: Responsive design with smooth transitions and persistent state management.

## 🛠️ Technology Stack
- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express, PostgreSQL (Neon Cloud).
- **Authentication**: Secure session-based authentication with high-level encryption.
- **Tracking**: Custom-built anonymous visitor logging system.

## 🌟 Vision
To empower the leaders of tomorrow through transformative educational experiences and innovative technology solutions.

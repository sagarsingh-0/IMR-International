# IMR International - Digital Education Portal
## Project Topic: Consent-Based Anonymous User Behavior Monitoring System

This project is a high-performance web application designed for IMR International. It features a custom-built **Consent-Based Anonymous User Behavior Monitoring System**, developed as a major project contribution to ethical web development and privacy-preserving analytics.

## 🔭 Research Focus
The core of this system is designed to solve the conflict between user privacy and data-driven insights. It implements:
- **Zero-Invasive Policy**: No tracking occurs until explicit consent is granted.
- **Anonymized Data Chains**: Uses SHA-256 one-way hashing for IP identifiers to ensure users remain anonymous.
- **Transparency Dashboard**: An admin portal to monitor high-level behavioral trends without exposing personal identifies.
- **Consent Lifecycle**: Real-time acceptance and withdrawal of tracking via a dynamic React-based UI.

## 🚀 Key Features
- **Educational Portal**: Full course discovery, program details, and admission management.
- **AI-Powered (Local) Logic**: Rule-based intelligent assistant for instant student support.
- **Dynamic Content**: Fully manageable administrative layers for tracking analytics and enquiries.
- **Premium UX**: Built with Framer Motion for smooth, high-end transitions.

## 🛠️ Technology Stack
- **Frontend**: React 19 (TypeScript), Vite, Tailwind CSS.
- **Backend**: Node.js, Express.
- **Database**: PostgreSQL (Neon Cloud) with Drizzle ORM.
- **Security**: Secure session-based auth and bcrypt encryption.

## 🏛️ Vision
To provide a secure, transparent, and ethically sounds digital infrastructure for modern educational institutions.
